# 项目背景
 
**一直在使用NET的webapi开发接口，今天终于有时间把一些常用的接口层场景封装一下，第一期封装的功能如下： 
1、封装统一的数据返回规范 
2、封装统一验证的过滤器 
3、封装部分字段验证的过滤器

# 项目说明
项目KunchiLibrary类库，负责各种验证的过滤器。
项目WebApi，演示了类库中的过滤器功能。
